Визализатор гипотез из теории чисел

Планируется реализация программы с графическим интерфейсом, считывающей математическую гипотезу и проверяющей её на маленьких числах, приводящей (при наличии) список контрпримеров, строющей графики и тп.

Авторы проекта: Нагуманов Юсуф (Б05-527), Ускова Алиса (Б05-527)

https://github.com/orengo42/Py_Project

Итерация 1 (фичи): Юсуф: Проверка гипотез, Реализация теоретикочисловых функций. Алиса: Реализация графиков и гистограмм.

Итерация 2 (фичи): Юсуф: Внутренний язык для выражений и его парсер. Алиса: Графический интерфейс приложения, выбор режимов и кастомная клавиатура

Библиотеки: Для математической логики sympy Для интерфейса Pyside6 Для работы с данными и парсера вероятно будеь использоваться csv и json
